## A Propos
DraftBot est le bot officiel du [serveur discord](https://www.draftman.fr/discord) du développeur DraftMan.
Il est développé en NodeJS et contient de nombreux modules :
- Discord.js
- Youtube API
- Fortnite API
- Twitter API
